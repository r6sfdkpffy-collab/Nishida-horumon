# Nishida Horumon Website

Sitio web estatico para にし田ホルモン / Nishida Horumon en Kyoto.

## Publicacion en Netlify

- Build command: dejar vacio
- Publish directory: `.`
- Sitio estatico: `index.html`, `styles.css`, `script.js` y `assets/generated/`

## Archivos incluidos

La version final usa solamente las imagenes generadas en `assets/generated/`.
La carpeta `assets/photos/` queda ignorada para no subir fotos reales que no se estan usando.
